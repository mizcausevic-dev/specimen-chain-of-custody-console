# Security

`specimen-chain-of-custody-console` includes an offline analyzer, CLI, and public synthetic operator dashboard for specimen-custody posture. It does **not** store lab credentials, patient identifiers, or authenticated write paths.

## Reporting

- [Open a security advisory](https://github.com/mizcausevic-dev/specimen-chain-of-custody-console/security/advisories/new)

If you need to share reproduction steps privately before filing an advisory, use the contact path already published on the main Kinetic Gain surfaces and avoid including secrets or real specimen data.
