This repo demonstrates the specimen-custody shape of Kinetic Gain Embedded:

- ingest a specimen-custody packet
- analyze labeling, temperature, handoff, storage, consent, and release posture
- expose the same logic as a public proof surface and an embeddable operator module

For embedded product work, the same primitive becomes an in-app pathology logistics and custody console where operators need read-safe evidence without direct access to raw laboratory systems.
